Project Story: Job Listings Analysis

Introduction

In this project, we aimed to analyze job listings for a specific search word to understand the requirements and technologies sought by employers. We used various libraries and technologies to achieve this goal, and we also explored the concept of PageRank to rank the pages we retrieved.

Technologies Used

We leveraged several interesting technologies throughout the project:

  • Requests library: This library was used to send HTTP requests and retrieve HTML pages containing job listings.
  • BeautifulSoup library: We utilized this library to parse and process the HTML pages returned by the requests.
  • Pandas library: This library was essential for saving the extracted data into DataFrames, which were later exported to CSV files for further analysis.
  • NLTK library: We used NLTK to process text, specifically to remove stop-words from the job requirements we extracted.
  • RE library: This library helped us find specific patterns in the text, such as extracting relevant keywords.

Runtime Analysis

We conducted two main queries:
For the first query - the runtime was approximately 1.5 minutes.
For the second query - the runtime for a single execution ranged from 1.5 to 2 minutes.

The primary factor affecting the runtime was bypassing LinkedIn’s bot detection and the time required to send various requests.

The processing time was negligible, taking only a fraction of a second to extract text, process it, save it in a DataFrame, and export it to a CSV file; with a fast internet connection and proximity to LinkedIn’s servers, the runtime could potentially be reduced to less than a minute.

Page Structure and Links

Our system returned two types of queries:

  • Query 1: The pages returned were mainly 48 independent pages. We sent 48 requests that weren’t connected, resulting in no hubs or authorities.
  • Query 2: The pages returned included one hub page containing 60 links to the first 60 job listings. Each job listing was a stand-alone page, making these the authority pages, with 60 in total.

PageRank Calculation with Random Walks
